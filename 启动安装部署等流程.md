数据库启动：
进入wsl的乌班图中
- sudo service postgresql start 如果乌班图有密码这时候会提示输入密码
- sudo service redis-server start
这两条就可以启动数据库postgresql和redis；

启动后端go服务：
当前命令是 go run ./cmd/server
- 启动目录是 D:\guoziji\apirouter\subiohub\backend
- 以后你自己启动时，直接执行：
cd D:\guoziji\apirouter\subiohub\backend
go run ./cmd/server
```
启动前端next-web服务（非wsl环境，我的在本机windows上）：
- 前端启动命令是(进到对应目录中)：
```
cd D:\guoziji\apirouter\subiohub\next-web
npm run dev
```
- 这次后端日志里有两条 WARN ：
  - TOTP encryption key auto-generated
  - security.url_allowlist.enabled=false
- 这两条更像开发环境提醒，不是启动失败，服务已经正常跑起来了。
